Integration of a Figma Design for a Web Application: Freelance Mission for a French Client

Made in Novomber 2023

About This Project:

I recently undertook a freelance project involving the integration of a Figma design for a French client. The task was to precisely convert multiple Figma pages into an interactive application website, emphasizing fidelity to the original design.

Key Features:

  • Responsive design to ensure an optimal viewing experience on all devices.
  • Integration of Figma mockups while adhering to all specifications using HTML, SCSS, and JavaScript
  • Using SCSS, each component such as buttons, logos, etc., is defined through specific functions incorporating variables for colors, mixins for typography, and other elements ensuring a modular and consistent approach.

Technologies Used:

HTML

SCSS

JavaScript
